실시간 음성 전사 (ASR)

Web Speech API 표준을 활용하여 화자의 음성을 즉각적으로 텍스트화합니다. 전용 서버 연결 없이도 지연 없는 전사 환경을 제공합니다.

Gemini AI 문맥 보정

발음이 모호한 전문 용어나 오타를 Gemini 2.0 Flash 모델이 회의 맥락에 맞게 지능적으로 교정하여 가독성을 높입니다.

AI 실시간 Q&A

회의 중 나오는 질문을 AI가 자동으로 감지하고, Google 검색 기능을 활용하여 신뢰할 수 있는 답변과 출처 링크를 즉시 제공합니다.

지능형 회의 요약

회의가 진행되는 동안 실시간으로 핵심 내용을 요약합니다. 결정 사항과 액션 아이템을 놓치지 않도록 체계적으로 정리해 드립니다.

발화자 프로파일링

음성 특징 분석을 통해 발표자와 참석자를 자동으로 구분합니다. 누가 어떤 발언을 했는지 명확하게 기록에 남깁니다.

HTML 스마트 리포트

완성된 회의록은 가독성 높은 HTML 리포트로 즉시 다운로드 가능합니다. 보관과 공유가 용이한 전문적인 포맷을 제공합니다.
