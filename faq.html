Frequently Asked Questions

VORA 서비스 이용에 대해 궁금한 점을 확인해 보세요.

서비스 일반
정말 회원가입이 필요 없나요?
네, 맞습니다. VORA는 사용자의 번거로움을 최소화하기 위해 별도의 회원가입이나 로그인 절차를 두지 않습니다. 웹사이트에 접속하는 즉시 모든 기능을 무료로 사용하실 수 있습니다.
서비스 이용 요금이 어떻게 되나요?
VORA는 100% 무료로 제공됩니다. 브라우저 내장 기술과 사용자의 개인 API 키를 활용하는 구조이기 때문에, 별도의 구독료나 사용료를 당사에 지불하실 필요가 없습니다.
보안 및 프라이버시
회의 내용이 서버에 저장되나요?
아니요. VORA는 'Serverless Architecture'를 지향합니다. 사용자의 음성 및 전사된 텍스트 데이터는 브라우저의 메모리상에서만 존재하며, 페이지를 새로고침하거나 닫는 즉시 소멸됩니다. 당사의 서버에는 어떠한 회의 기록도 남지 않습니다.
입력한 Gemini API 키는 안전한가요?
사용자가 입력한 API 키는 본인의 브라우저 로컬 스토리지(LocalStorage)에만 암호화되어 저장됩니다. 이 키는 당사 서버로 전송되지 않으며, 오직 Google Gemini 서버와 통신할 때만 사용됩니다.
기술 및 성능
어떻게 전문 용어를 그렇게 정확하게 보정하나요?
두 가지 핵심 로직이 작동합니다. 첫째, 사용자가 설정한 **'우선 인식 용어'**를 기반으로 1차 필터링을 거칩니다. 둘째, **'지능형 피드백 루프'**를 통해 실시간으로 생성된 회의 요약본을 AI에게 다시 전달합니다. AI는 이 요약본을 토대로 현재 회의의 전문적인 맥락을 이해한 상태에서 전사 결과를 교정하기 때문에, 단순 전사 도구보다 압도적으로 높은 정확도를 보여줍니다.
AI가 회의의 전체 흐름을 어떻게 기억하나요?
긴 회의의 모든 텍스트를 AI에게 계속 보내면 처리 속도가 느려지고 비용이 상승합니다. VORA는 이를 해결하기 위해 **'요약 재주입(Summary Re-injection)'** 기술을 사용합니다. 주기적으로 생성된 핵심 요약 데이터를 AI의 '단기 기억'으로 활용함으로써, AI는 수 시간의 회의 중에도 일관된 맥락을 유지하며 질문에 답변하고 오타를 수정할 수 있습니다.
인식 언어는 어떻게 변경하나요?
현재 한국어, 영어, 일본어, 중국어를 공식 지원합니다. 다국어 회의 환경에서도 설정창에서 언어를 변경하여 유연하게 대응할 수 있습니다.
AI 답변이 생성되지 않을 때는 어떻게 하나요?
Gemini API 키가 올바르게 입력되었는지 확인해 주세요. 또한, Google의 Rate Limit(호출 속도 제한) 정책에 의해 일시적으로 대기가 발생할 수 있습니다. 관리자 제어센터에서 'Gemini API Queue' 상태를 확인해 보시기 바랍니다.
