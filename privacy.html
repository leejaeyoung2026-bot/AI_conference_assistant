개인정보처리방침

본 서비스는 사용자의 개인정보를 소중히 다루며, 관련 법규를 준수합니다.

1. 수집하는 개인정보

본 서비스는 회원가입 절차가 없으며, 별도의 개인식별정보를 서버에 저장하지 않습니다. 사용자가 입력하는 Gemini API 키는 브라우저의 로컬 스토리지에만 저장됩니다.

2. 데이터 처리 방식

음성 데이터는 브라우저 내 세션 동안만 처리되며, 분석을 위한 텍스트 데이터는 Google Gemini API로 전달됩니다. 이 데이터는 서비스 제공 목적 이외에 사용되지 않습니다.

3. 쿠키 사용

사용자 설정 유지를 위해 브라우저 로컬 스토리지를 사용합니다.
