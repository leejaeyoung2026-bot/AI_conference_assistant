개인정보처리방침

본 서비스는 사용자의 개인정보를 소중히 다루며, 관련 법규를 준수합니다.

1. 수집하는 개인정보

본 서비스는 회원가입 절차가 없으며, 별도의 개인식별정보를 서버에 저장하지 않습니다. 사용자가 입력하는 Gemini API 키는 브라우저의 로컬 스토리지에만 저장됩니다.

2. 데이터 처리 방식 및 보안

음성 데이터는 브라우저 내 세션 동안만 실시간으로 처리되며, 분석을 위한 텍스트 데이터는 Google Gemini API로 전달됩니다. 데이터 보안 수준은 사용자가 입력한 API 키의 플랜에 따릅니다.

  • 유료 플랜(Paid Tier) 사용 시: Google의 정책에 따라 입력된 데이터와 출력된 결과물은 모델 학습에 사용되지 않으며 기업용 수준의 보안이 유지됩니다.
  • 무료 플랜(Free Tier) 사용 시: Google의 서비스 품질 개선 정책에 따라 입력된 데이터가 익명화되어 검토되거나 모델 학습에 활용될 수 있습니다.

본 서비스는 개인의 선택에 따른 API 키 사용을 존중하며, 어떠한 회의 데이터도 본 서비스의 자체 서버에 저장하지 않습니다.

3. 쿠키 사용

사용자 설정 유지를 위해 브라우저 로컬 스토리지를 사용합니다.
